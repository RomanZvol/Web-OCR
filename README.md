# Web-OCR

Зручний додаток для OCR (зчитування тексту з фото).

## Стек технологій
- React
- TypeScript
- Vite

## Запуск проекту
1. Встановіть залежності: `npm install`
2. Запустіть dev-сервер: `npm run dev`
